# Fonts — Neue Montreal

Download the Neue Montreal font family from Fontshare:
  https://www.fontshare.com/fonts/neue-montreal

Place the `.woff2` files in this directory with these exact names:

  NeueMontreal-Light.woff2
  NeueMontreal-Regular.woff2
  NeueMontreal-Medium.woff2
  NeueMontreal-Bold.woff2
  NeueMontreal-Italic.woff2
  NeueMontreal-MediumItalic.woff2

The font files are loaded in `app/layout.tsx` via `next/font/local`.
Until they're added, the site will fall back to Inter → system sans-serif.
